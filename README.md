# Student Management System

A full-stack web application for managing student records, attendance, courses, and academic data using modern web technologies.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MySQL

## Features
- Student registration and record management
- Course and attendance management
- Role-based access (Admin, Teacher, Student)
- CRUD operations
- Secure and centralized data storage

## Setup Instructions
1. Clone the repository  
   git clone https://github.com/HarshCS23034/student-management-system.git

2. Install dependencies  
   npm install

3. Configure MySQL database credentials

4. Start the server  
   npm start

5. Open browser  
   http://localhost:3000

## Purpose
This project is developed for learning and academic purposes to demonstrate full-stack web development using Node.js and MySQL.
